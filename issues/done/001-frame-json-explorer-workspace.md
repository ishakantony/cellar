## What to build

Frame the JSON Explorer as a rounded, layered workspace card that visually matches the Vault dashboard while preserving the existing editor/tree split workflow. This slice establishes the outer shell, available-height behavior, inset pane treatment, subtle resizable divider, desktop split layout, mobile stacked layout, pane-ratio persistence, and non-persistence of editor contents described in the parent PRD.

## Acceptance criteria

- [ ] The JSON Explorer renders inside a rounded workspace card instead of feeling full-bleed.
- [ ] The editor and tree panes visually read as inset panels inside one cohesive workspace.
- [ ] The split-pane divider remains present, subtle, and resizable on desktop.
- [ ] The existing mobile editor-over-tree stacked layout is preserved.
- [ ] The page continues to use the available shell content height for editor usability.
- [ ] The existing pane-ratio persistence behavior remains intact.
- [ ] The editor text remains unpersisted across refreshes.
- [ ] Tests validate user-visible structure and preserved behavior without asserting brittle styling class names.
